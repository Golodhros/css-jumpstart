Terminal Block Elements

Headings, paragraphs, blockquotes, definition terms, addresses, and table captions are terminal block elements. They may contain only content. An HTML validator will declare a document invalid if you attempt to put block elements inside terminal blocks.

A blockquote is a terminal block.
NOTE:
The content of terminal blocks is always inline.
An address is a terminal block.
Table caption is a terminal block.
